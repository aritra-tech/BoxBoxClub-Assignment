# BoxBoxClub Assignment

This is an Android application built using Jetpack Compose that displays Formula 1 driver information, upcoming race sessions, and other related content.

## Features

*   **Driver Information:** View details about Formula 1 drivers, including their position, wins, and points.
*   **Upcoming Races:** Get information about the next race session.
*   **Educational Content:** Access articles and guides about Formula 1.
*   **Modern UI:** A clean and modern user interface built entirely with Jetpack Compose.

## Tech Stack

*   **UI:** Jetpack Compose
*   **Architecture:** MVVM (Model-View-ViewModel)
*   **Asynchronous Programming:** Kotlin Coroutines
*   **Networking:** Ktor
*   **Dependency Injection:** Hilt (or specify if another library is used)

## How to Build

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/BoxBoxClubAssignment.git
    ```
2.  Open the project in Android Studio.
3.  Let Android Studio download the required Gradle dependencies.
4.  Build and run the application on an emulator or a physical device.
